Drone Photography, 3D Printing services, Electronics Trainings/Workshops and more ... by Goan FPV

We are a family shop based out of South Goa, India, with a focus in providing service related to Drones/Custom Quadcopter videography. And also provide a 3D printing services using FDM(Fused deposition modeling) printers using PLA and ABS materials.

In addition, we provide 3D printing and basic electronics training and workshops for-

  • 3D Printing and Design
  • Learn basic electronics using Arduino
  • Learn programming using the Raspberry Pi
  • Python jumpstart course
